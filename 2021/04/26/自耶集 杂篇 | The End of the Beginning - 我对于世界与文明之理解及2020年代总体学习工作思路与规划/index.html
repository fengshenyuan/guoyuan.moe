《自耶集》 杂篇 | The End of the Beginning - 我对于世界与文明之理解及2020年代总体学习工作思路与规划

发表于 2021-04-26 更新于 2021-06-19 分类于 自耶集

导言
到2022年底，未来两年内再细读学习20本书的内容，精读30本书作为其补充，另50本书作为扩展和丰富，将自己知识文化的宽度和深度都提升一两个台阶。这些书不是为现在读的，而是为十年后读的。就如同小时候读的儒道释一样，那时用不到，但现在就自然而然，水到渠成地用到了。

那么十年后有哪些能力是被强力需要的呢？

  • 关于判断一个方向是否值得投入去做的决策能力
  • 关于如何从零搭建一个团队、运营一个产品的能力
  • 关于如何从零学习一门新的技术、攻克一个技术难题、转化一项科研成果为实际产品的能力
  • 关于一眼之凭的逻辑推断能力
  • 关于理解国家决策方向、把握国家机遇方面的能力
  • 清楚地记忆与理解历朝历代历史变化，全国各地地域文化、民族特色的能力
  • 清楚地理解国际关系、国际贸易、现代文明格局的能力
  • 对于本国传统文化的深层次的理解、掌握、运用的能力
  • 对于本国国内情况的全面且切实的理解
  • 对于本国年轻一代的理解
  • 精神内守、盈余输出的能力

有哪些是值得现在就播下种子，到时候自然就会郁郁葱葱的事情？

  • 现在开始写作，技术性写作和非技术性写作都需要尝试起来
  • 现在开始需要积极参与新技术、新项目中去，参与到社区的构建、转化事情中去
  • 阅读更多的关于国内山川地理、风土人情的资料，走访全国更多的地方

对于世界的学习与理解
对于世界的学习与理解的几个阶段
第一步 阴虚阳实，饱览天下
第二步 智识圆融，妙用无碍
第三步 变化窥尽，知所本末；般般打破，自然无疑

现在还没有走完第一步，第一步大概也可以分为以下3个小阶段

  • 基础教育阶段 了解和理解一些关于这个世界的基本事实，构建起基本的认知框架
  • 基础教育之后的分野
    • 专业教育 在基础教育之上，更加全面和深入的了解、掌握这个世界某一个方向的知识和内容
    • 高等教育 在基础教育之上，更加全面和深刻的了解和理解这个世界的多样性、丰富性、专业性以及多种事物内在的普遍联系和对应的基本理论
  • 在高等教育之后经过5~10年的实践使自己对于理论和实际的理解达到一个统一的新高度，最终可以说自己对于这个世界相当一大部分的事情都有过接触和思考，而不是懵懵懂懂、无知和理解无能，最终达到第一步的阴虚阳实，饱览天下的状态。

自己的智识状态大概经历了从乡村小世界观，到古典大世界观，再到初步现代化世界观的转变，目前正处于扩展自己视野，将更多的领域与知识纳入到自己知识能力结构中来阶段，即现代化世界观的中期阶段。该阶段的主要任务和目标仍然是学习和理解，虽然这事情枯燥无聊且持续了几十年了，但这就是事实，而且这事情似乎没有尽头。这事你不得不做，这路你也不得不走！

文明之理解
以文明系统视角来看，两条线，几个方向：

  • 文史哲：世界史/文明史/科技史/哲学史(系统发展史) + 当代经济、政治、国际形势以及地理(系统当前面貌概览)
  • 数理生：现代数学、现代物理、现代生物医药、现代工程制造(能源、交通、城市、太空探索、生物医药、材料与装备制造) - 系统迭代驱动力系统(现代系统内核) - Refactor趋势及其方法论

将整个文明置于一般系统发展观之下来进行解析与认知，让自己获得超越自身局限性的视角与观察力，明白所谓虚实与艺术之间的区别和分野，明白这到底是一套怎样的系统，依靠何种规则而运行，有怎样的特性，其基本模型如何，其底层依赖如何，其发展趋势如何，其局限性如何，其发展上限如何，其可知者如何，不可知者如何，其可控制者如何，不可控制者又如何？

一个极有趣之话题：中国文明之内核 - 因为目前世界处于双文明内核碰撞与融合的初级阶段(Unix vs Windows)，关于整套系统之未来，中国文明将扮演何种角色，其自身内核如何演变，其独立性在何种程度上还能够得以保留，这都是极为有趣的话题。另，如果想要整套系统保持多内核共存的模式，又该采取何种之有效策略，又是一极有趣之话题。

基本模型：人(有限生命节点) -> 主观能动性网络 -> 有限生命体迭代文明 -> 区域文明 -> 星球文明 -> 星际文明

文明根植于宇宙之内核之上，文明的内核是对宇宙内核的二次包装与剪裁，宇宙之内核其核心在于数学与物理，即文明内核视角所谓之纯数学与理论物理，文明之内核对于宇宙内核的包装产生应用数学和应用物理及其他工程与应用学科，并促进文明之生长！
忽生忽灭，谁生谁灭？
宇宙恒其久，文明多生灭。
个体节点的failure，即个体有生有灭是文明的基本事件(所谓基本事件，即所谓每时每刻都在发生，且文明本就建立于这些基本事件的发生之上，故个体的有生有死对文明来说毫无关系，甚至个体有生有死才是文明得以存续的核心关键)
区域文明的分区、主动关闭是文明演化的正常事件
这一思想可为吾思想之准绳，可以之绳墨是非前人之学术文章、思想观念，并以此为线，寻找古人中值得推崇者，寻古人中之知己，寻古人中师友，看他们是如何看待与理解这个世界，又如何度过自己的一生。

    坚持一个核心要点：不要随波逐流，不要亦步亦趋，不要局限于别人所珍爱的细节之中，要坚持走自己的核心道路，始终坚持构建、不断完整和圆满自己的理论体系。要对别人的理论有判断，有选取！

现实与有趣的问题
如何建设一个具有全球辐射能力的核心区域文明体系？- 星球文明
集世界总体之文明，创世界总体之进步。

所思考者，文明需要怎样的空天能力？
当前地球之星球文明所缺乏者：空天能力。地面之能力，经几千年文明之发展，已足够支撑使用。然而，将所有技术应用于空间及空天之上，获得更为准确、庞大的信息、能源、特殊的物理环境对整个文明进行补强，尚做得不够！利用空天能力
(1). 可以对星球文明形成第一层主动保护罩，防护地球受到外太空灾难影响，或提前报警；
(2). 可以利用太空环境对宇宙或地球获得更为详细的观察，获得极为有用的空间、地理信息；
(3). 可以利用太空卫星系统形成全球通信系统，提供全球性定位、事件分析系统，从根本上提高全球协作能力；

  • 教育、科研(MATLAB/Mathematica等全球通用软件，面向全球的哲学思考体系)
  • 贸易(制造业、金融业)
  • 文化输出、地区性事务协调
  • 面向全球的产品设计、技术研发体系
  • 面向全球的综合信息服务能力(北斗全球定位系统)

全球文明及其核心的区域文明，总体上是合作与竞争关系

当前全球文明的四大话语体系：

  • 以美国为首的美国话语体系
  • 以德法(英)为首的欧洲文明体系
  • 以俄罗斯为首的中东话语体系
  • 以中国为首的亚非拉话语体系

这四大体系发展出了相对独立的文明体系，且在一定程度上在海陆空等各方面具有辐射全球的影响力，几乎可以当做是整个文明的某种备份。如果一个出现问题，另外的都能很快的接过或继承其话语体系，几乎可以说全球文明是这四大文明体系相互开放交流所呈现的整体关系而已。当然，在这其中也还有其他一些具备发展出独立话语体系潜力的国家，比如日本、印度等，但现在还不足为虑，他们在已有的四大文明体系面前还不能构成有效的全球影响力。

在开放交流的全球文明体系之上，我们逐步形成了一些全球性的基础设施，也即全球性的文明遗产。这些遗产包括诸如
联合国 - 在一些全球事务上的协作体系

  • 全球性协作交流的语言 - 阿拉伯文、中文、英文、法文、俄文和西班牙文
  • 全球性网络通信工具 - 全球光缆等Internet等信息技术生态 + 3G-4G-5G等移动通信
  • 全球性交通网络能力 - 当前仍然以航空业为主 + 海运 + 航天合作
  • 全球性学术教育体系 - 全球性科学技术协作体系，全球性科研互通互做
  • 全球性制造业体系 - 全球性产业链分工合作体系
  • 全球性金融贸易体系 - 当前以美元为首的国际金融体系(不够好 - 没有竞争对手)
  • 全球性能源供给体系 - 以中东、俄罗斯为首的国际化石能源供应体系(不够好 - 没有竞争对手)
  • 全球性航天探索 - 目前每个独立的文明体系都有自己相对独立的航天探索项目和规划(比如卫星导航系统) - 比较健康
